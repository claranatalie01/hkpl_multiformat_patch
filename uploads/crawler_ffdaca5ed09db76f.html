In the course of the implementation of the "Smart Library System" project in phases for the Hong Kong Public Libraries, new self-service equipment with application of radio frequency identification (RFID) technology will be gradually deployed to different libraries for provision of related services. The first phase of new equipment includes Self-checkout Machine, Self-service Kiosk, Smart Book Drop and Smart Book Return.

    1. Self-checkout Machine

      Desktop type Self-checkout Machine Floor standing type Self-checkout Machine
      The new Self-checkout Machine uses RFID technology to handle the checkout process. Readers can now checkout multiple items in one go, without the need of scanning the barcode of the items to complete the checkout process, providing a new self-service experience for the readers.

      There will be additional features available on the Self-checkout Machine after the full launch of the "Smart Library System" in future, including payment function and login by QR code scanning via mobile app.

      Tips for Using the Self-checkout Machine
    2. Self-service Kiosk

      The Self-service Kiosk, in the first phase, will provide functions on searching of library catalogs, including browsing of library website and using of the Digital Collection. The Self-service Kiosk is also equipped with a wide touch screen for enhancing user experience.

      After the full launch of the "Smart Library System" in future, the Self-service Kiosk will provide additional services, which include document printing, enrollment of extension activities, interactive enquiry service (Chatbot), library account management and payment function, etc.

    3. Smart Book Drop and Smart Book Return

      Starting in late November 2025, the Hong Kong Public Libraries has progressively introduced the Smart Book Drop and Smart Book Return at public libraries across all districts. These self-service return facilities utilise Radio Frequency Identification (RFID) technology to process the return of library materials in real time and update borrowers’ loan records instantly.

      All library materials returned via the Smart Book Drop or Smart Book Return will be processed and recorded by the library system in real time. Fines incurred for overdue items will also be calculated and recorded in the library system based on the actual return time, with midnight as the start of each day.

      The Smart Book Drop or Smart Book Return, located outside the library, operates according to the hours of the premises where it is situated, not the library’s operating hours. For details regarding the smart return facilities and their service hours at respective public libraries, please refer to the service information for individual libraries published on the website:
